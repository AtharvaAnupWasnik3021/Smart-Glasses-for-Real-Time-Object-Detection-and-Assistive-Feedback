# Smart-Glasses-for-Real-Time-Object-Detection-and-Assistive-Feedback
This project aims to design smart glasses that detect objects using computer vision and transmit sensor data wirelessly for assistive analysis.


Objectives:
To implement real-time object detection using a USB camera and Raspberry Pi.
To transmit detected object data via Wi-Fi to a connected laptop for visualization.
To integrate ESP32-based sensors (IR, Ultrasonic, Touch) for environmental awareness.

Components:
Raspberry Pi 3B+
ESP32 Development Board
USB Camera
IR Sensor
Ultrasonic Sensor
Touch Sensor
Wi-Fi Module (built into ESP32)
Power Bank (10,000 mAh)
Connecting Wires
